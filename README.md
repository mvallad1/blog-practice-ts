Create Blogs:
Add a new blog with

Title

Description

Image (URL)

Date Created

Edit Blogs:
Update any part of a blog (title, description, image, or timestamp).

Remove a blog entirely from the list.

The UI updates immediately when you add, edit, or delete blogs.
